HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • "ul" tag creates unoredered list of items, we can use "li" tags to places bullet points into our list just like being used for these notes
  • When organizing content for the user creating "section"s inside the "body" of the code are important for orgnizing information just like we are doing here with the different sections of code and adding "id" or names to our sections

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch you are currently on
  • git pull: pulls you back a branch to the branch above
  • git checkout -b "enter branch-name" creates a new branch and transfers you to that branch
  • git checkout "branch-name" is another way to transport through branches
  • add -A is used to commit all files in the working branch

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
